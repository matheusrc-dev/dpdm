## Criação de um projeto Expo e Workflow no Git

- Criar o projeto com o comando `yarn create expo --template`
- Enviar o projeto para o git
- Criar uma issue para as configurações iniciais
- Criar um projeto na aba "projects" do github
- Vincular a issue na aba "projects" criada anteriormente
- Fazer um checkout para `git checkout -b @matheusrc-dev/{issue_id}/branch-name`
- Fazer o push dessa branch para o respositório
- Criar uma pull request
- Linkar a pull request com a issue criada anteriormente
- Fazer o merge da pull request com a opção 'Squash and Merge'
- Verificar se a isseue foi fechada automaticamente
